🎵 Spotify Clone

A simple Spotify-inspired music streaming interface built to enhance my frontend web development skills. This project was created as a hands-on learning experience to understand how modern music streaming platforms are designed and how interactive web interfaces are developed using core web technologies.

📖 About the Project

The goal of this project was to recreate the look and feel of Spotify while practicing responsive layouts, UI design, and JavaScript-based interactions. Through this project, I gained practical experience in building a clean, user-friendly interface and organizing frontend code effectively.

This project focuses on improving frontend development concepts rather than replicating every feature of the original Spotify application.

🎯 Objectives

- Practice frontend web development fundamentals.
- Improve HTML, CSS, and JavaScript skills.
- Build a responsive and visually appealing user interface.
- Understand webpage structure and user interaction.
- Learn project organization using Git and GitHub.

 ✨ Features

- Spotify-inspired homepage design
- Responsive user interface
- Navigation menu
- Music cards and playlists
- Music player section
- Interactive buttons and hover effects
- Clean and organized layout

 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
